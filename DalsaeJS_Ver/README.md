# Dalsae
달새 MK-III vue-electron으로 제작했어요!
* npm install 후 진행 해주세요.
* \src\renderer\APIKey.js 파일 생성 후 ConsumerKey(App 공개키), ConsumerSecretKey(App 비밀키)를 export 해주셔야 프로그램이 동작 합니다!
* APIKey.js의 내용입니다.
* export default{
  ConsumerKey : '퍼블릭키',
  ConsumerSecretKey : '비밀의키'
} 


달새 단축키 목록입니다.
* 1: 홈 화면
* 2: 멘션 화면
* Home: 맨위로
* End: 맨아래로
* Space: 홈/멘션화면 더 불러오기
* c: 대화 불러오기
* Backspace: 대화 숨기기
* A: 전체 답변
* R: 답변
* F: 관심글
* T: 리트윗
* G: 이미지 열기
* V: 메뉴 열기
* U: 트윗 입력칸 가기
* Delete: 트윗 삭제
* X: 인용트윗 보기
